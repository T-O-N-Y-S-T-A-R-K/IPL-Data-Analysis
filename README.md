# 🏏 IPL Data Visualization and Analysis

This project performs **data analysis and visualization** on Indian Premier League (IPL) datasets using Python. It provides insights such as match wins by team and sets up the environment for deeper machine learning-based analysis.

---

## 📁 Datasets Used

1. **`matches.csv`** – Contains match-level information like teams, winners, venues, and match results.
2. **`deliveries.csv`** – Contains ball-by-ball delivery data including batsmen, bowlers, runs scored, and dismissals.

---

## 📊 Features

- ✅ Static visualization using **Matplotlib + Seaborn**
- ✅ Interactive visualization using **Plotly** (opens in browser)
- ✅ Initial setup for machine learning models using **scikit-learn**
- ✅ Feature visualization tools using **Yellowbrick**

---

## 📷 Sample Visualizations

- **Bar plot** of match wins by each team using both Matplotlib and Plotly.
- Future-ready for:
  - Toss analysis
  - Venue impact
  - Player performance metrics
  - Feature importance and dimensionality reduction
